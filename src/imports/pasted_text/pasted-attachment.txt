Design a complete mobile app UI for a geriatric exercise application for older adults.

The app helps seniors improve mobility, strength, balance, and daily movement using guided exercises.

The design must be very simple, accessible, and senior-friendly with:
large buttons
large text
clear navigation
soft colors
rounded cards

App flow:

1 Welcome screen
Show the app name
Short message about improving mobility
Buttons:
Start
I am new
I have used the app before

2 Onboarding questionnaire
Create a multi-step questionnaire with around 15 questions about:
age
gender
mobility level
pain areas
balance
ability to walk
daily activity
exercise goals
energy level
risk of falling

3 Main dashboard
Top header with app name
A large card called:
Recommended for you

Below that create four categories:
Legs and glutes exercises
Core exercises
Arm exercises
Mobility and flexibility exercises

4 Exercise category screens
Each category should show a list of exercises in cards.

Example exercises:

Legs and glutes:
March in place assisted
Heel raises
Toe raises
Sit to stand
Hip abduction
Hip extension
Mini squats
Step forward
Single leg balance
Glute bridge

Core:
Abdominal bracing
Pelvic tilt
Diaphragmatic breathing
Dead bug basic
Bridge
Modified plank
Seated trunk rotation
Side trunk bending
Trunk extension
Bird dog modified

Arms:
Front raise with bottles
Side raise seated
Biceps curl
Wall push ups
Shoulder rotations
Ball squeeze
Arm raise no weight
Resistance band row
Arm extension

Mobility:
Stretching routine
Wrist and fingers mobility
Hip mobility
Neck mobility
Ankle mobility

5 Exercise detail screen
Show:
exercise name
difficulty
pain level
importance
daily life benefits
sets and repetitions
rest time
video placeholder
Start exercise button
